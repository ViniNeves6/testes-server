Histórico de Publicações

  • UX-Tracking: uma ferramenta para avaliação de experiência dos alunos no contexto das aulas EaD

    Em 2023, realizou-se um estudo de caso para avaliação UX em cenário educacionais com a UX-Tracking. Essa pesquisa utiliza das técnicas de rastreamento que a ferramenta disponibiliza para permitir a avaliação de experiência dos alunos em aulas EaD

  • Rastreamento de mouse para avaliação de experiência do usuário em portais de notícias: Um estudo de caso

    Outra pesquisa com o UX-Tracking, publicada em capítulo de livro, realizou um estudo de caso com a ferramenta para avaliação de UX em sites de notícias, como o portal do O liberal e DOL

  • Perspectivas do monitoramento de Experiência de Usuário em diferentes domínios com ferramenta de captura multimodal

    Com o desenvolvimento do UX-Tracking, ainda em 2022, foram realizadas diversos estudos com a ferramenta, um deles propôs um estudo exploratório para identificar e analisar as perspectivas de UX em diferentes domínios monitorados com o UX-Tracking.

  • UX-Tracking: Web and Multimodal Tool for User Experience Evaluation

    Ainda em 2022, a ferramenta T2-UXT evoluiu para o atual framework, denominado User Experience Tracking (UX-Tracking), uma ferramenta Web de captura multimodal que realiza o rastreio de mouse, olhar, teclado e voz para melhoria de UX em cenários web diversificados.

  • Avaliação de Experiência do Usuário de diferentes regiões do Brasil em sites de notícias: um estudo de caso Norte e Sul

    Em 2022, é realizada uma avaliação suportada pelo T2-UXT, comparando os dados obtidos por esta ferramenta com a opinião dos participantes desta pesquisa, através do Quis Teste (adaptado da heurística UX)

  • An Evaluation Framework for User Experience Using Eye Tracking, Mouse Tracking, Keyboard Input, and Artificial Intelligence: A Case Study

    Também em 2021, propõe-se um estudo de caso em sites com T2-UXT que emprega métodos de rastreamento, questionário de autoavaliação e algoritmos de IA para Avaliação de UX e categorizar usuários em termos de perfis de desempenho.

  • Analytical Model for Classifying Areas of Interest in Interactive Systems

    Ainda em 2021, propõe um estudo com T2-UXT em conjunto com um modelo analítico que define Áreas de Interesse do Usuário com base em objetos HyperText Markup Language (HTML) da interface e os classifica com base em um algoritmo T2-UXT para avaliação UX

  • Análise da Experiência do Usuário (UX) de Narrativa Transmídia através de Mouse-Tracking

    Em 2021, realizou-se a pesquisa utilizou a antecessora AIMT-UXT como método de avaliação da experiência do usuário a partir de métricas de rastreamento de mouse.

  • T2-UXT: A Tool to Support Transparency Evaluation in Software Ecosystems Portals

    Em 2020, a ferramenta evoluiu para o Tracking Techniques Based User Experience Evaluation Tool (T2-UXT), que realiza o Rastreio de mouse, olhar e teclado para melhoria de UX em diversos cenários de ecossistema de Softwares

  • User Experience Evaluation Using Mouse Tracking and Artificial Intelligence

    Ainda em 2019, com a ferramenta AIMT-UXT, foi realizado um outro trabalho de Rastreio de Mouse e Olhar voltado melhoria de UX, usando lógica Fuzzy em um estudo de caso

  • Metodologia Baseada em Agrupamento de Rastreamento de Mouse e Olhar para Melhoria da Experiência do Usuário

    Também em 2019, foi produzido mais trabalhos com o AIMT-UXT, ferramenta antecessora ao UX-Tracking, agora com Rastreio de Mouse e Olhar voltados para melhoria nas avaliações UX, usando os métodos k-means

  • Comparative Study of User Experience Evaluation Techniques Based on Mouse and Gaze Tracking

    Em 2019, foi introduzido uma ferramenta com o Artificial Intelligence and Mouse Tracking-based User eXperience Tool (AIMT-UXT), para coletar os dados de mouse-tracking, e SearchGaze.js, para a coleta de dados de rastreamento ocular e pontos fixos do olhar

  • Web Tracer: Aplicativo para monitoramento de rastreio do Mouse

    Em 2018, foi desenvolvido o Web Tracer, uma ferramenta de monitoramento e avaliação UX a partir das trajetórias e cliques do mouse, a qual, posteriormente, foi encapsulada no framework UX-Tracking
